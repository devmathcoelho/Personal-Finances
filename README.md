# Personal Finances

Uma aplicação web para controle financeiro pessoal construída em Angular, com foco em visualização de dados financeiros através de gráficos e uma interface simples e limpa.

## Descrição

Este projeto é um website para gerenciar finanças pessoais. A ideia é simplificar o registro de receitas, despesas e mostrar visualmente esses dados para que você entenda melhor para onde seu dinheiro está indo.

## Funcionalidades principais

- Registro de transações financeiras (receitas e despesas)

- Visualização de dados por meio de gráficos (dashboard financeiro)

- Interface responsiva para facilitar o uso

- Organização das finanças para planejamento e acompanhamento

## Tecnologias

- Frontend: Angular (TypeScript)
- HTML / CSS
- Bootstrap

## Como rodar localmente

```
Clone o repositório:

git clone https://github.com/devmathcoelho/Personal-Finances.git
cd Personal-Finances
```

##Instale dependências:

```
npm install

Rode a aplicação:
ng serve

Acesse no navegador:
http://localhost:4200
```

## Para gerar o build de produção:

ng build --configuration production

Os arquivos finais ficam em dist/.
